## 🚀 快速部署（3 步上线）

### 最简单方式（推荐）

**步骤 1：创建 GitHub 仓库**
1. 访问 https://github.com/new
2. 仓库名：`meimagic-website`
3. 设为 Public
4. 点击 "Create repository"

**步骤 2：上传文件**
1. 点击 "Add file" → "Upload files"
2. 拖入 3 个文件：`index.html`, `styles.css`, `script.js`
3. 点击 "Commit changes"

**步骤 3：启用 Pages**
1. Settings → Pages
2. Source: Deploy from a branch
3. Branch: main → / (root)
4. 点击 "Save"

**完成！** 1-2 分钟后访问：
```
https://YOUR_USERNAME.github.io/meimagic-website
```
